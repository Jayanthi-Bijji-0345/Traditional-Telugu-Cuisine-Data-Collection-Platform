Thank you for your interest in contributing to this project!  
Your support helps us preserve and digitize the culinary heritage of Telangana and Andhra Pradesh through open collaboration.

📌 Contribution Areas
We welcome contributions across the following areas:

Recipe Documentation: Collect and upload traditional Telugu recipes.

Audio/Video Uploads: Assist in logging multimedia formats.

UI/UX Improvements: Enhance user experience on the data collection portal.

Code Enhancements: Improve existing logic, fix bugs, or add new components.

Bug Reports: File detailed issues with steps to reproduce and screenshots if possible.

Documentation Updates: Help improve clarity, grammar, or structure in markdown files.

📐 Development Guidelines
Adhere to PEP8 for Python or appropriate coding standards for your language.

Include helpful comments for any non-trivial logic.

Test your changes thoroughly before submission.

Use relative paths and environment variables where needed.
